# VitaScrob
Logs tracks to scrobbler.log file for use in qtscrob natively on the built in psvita music app

# Installation
Installation goes trough taihen plugin system
1. Download the plugin
2. put the file on your PSVita in your "tai" folder
3. edit the config file and add this line to it:
   *NPXS10009
   ur0:tai/vitascrob.suprx
4. Reboot psvita

Your scrobbler.log should appear in Root/Data/Vitascrob/scrobbler.log
